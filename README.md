ระบบจองห้องประชุมด้วย **Python** ที่รองรับการทำงานทั้งแบบพื้นฐานและเวอร์ชันที่ใช้ **AI**  
ช่วยให้การจัดการห้องประชุมมีประสิทธิภาพมากขึ้น 🚀

---

## ✨ Features
- 📂 **BookingMeetingRoom.py** → ระบบจองห้องประชุมแบบพื้นฐาน  
- 🤖 **BookingMeetingRoomWithAI.py** → รองรับการรับ Input เป็นประโยคยาว ๆ และใช้ AI ช่วยประมวลผล  
- 🧠 **BookingWithAI_Full_Version.py** → เวอร์ชันเต็มของระบบ AI  
- ⚙️ **AI_optimization.py** → โมดูลสำหรับปรับปรุงการทำงานของ AI  
- 📊 **BMR-Update_Table.py** → อัปเดตตารางการจองห้องประชุม  
- 🔍 **HeuristicTest.py / HeuristicVersion** → ทดสอบ heuristic function สำหรับการเลือกห้อง  
- 🧪 **Meetingroom_test** → โฟลเดอร์สำหรับการทดสอบระบบ  

---

## 📖 Usage
- ▶️ Run BookingMeetingRoom.py → ใช้งานระบบจองห้องประชุมแบบพื้นฐาน
- 🤖 Run BookingMeetingRoomWithAI.py หรือ BookingWithAI_Full_Version.py → ใช้ระบบที่รองรับ AI
- ⚙️ ปรับแต่ง heuristic หรือ optimization ได้ผ่านไฟล์ที่เกี่ยวข้อง
